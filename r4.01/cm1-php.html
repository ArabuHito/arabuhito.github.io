...

A code snippet :
...

<?php
declare (strict_types=l );
function somme(int $x, int $y) : int
	return $x + $y;
echo somme(l .2, 3); // Ce code va déclencher une erreur.
?>

PHP n'est pas un langage typé, on déclare une variable en précédant son nom d'un $ et on y assigne une valeur.

Déclaration de variables en PHP :
$message = "Hello world!"; // Déclare un string
$int = 3; // Déclare un int
$reel = 15.3; // Déclare un double

PHP supporte l'encapsulation, les variables contenues dans une fonctions sont locales et seront supprimés de la mémoire une fois la fonction terminé. Bien entendu, une variable globale sera déclaré si elle est déclaré hors d'une fonction.

Variables statiques

Si on souhaite conserver une variable hors d'une fonction, on peut utiliser le mot clé static qui indique à l'interpréteur de ne pas détruire la variable une fois la fonction terminée.

String manipulations :
...

Puisqu'on travaille avec du web, ce ne serait pas drôle si on ne pouvait pas manipuler du texte, information première de ces magnifiques pages... (juste après les memes).

PHP a beau être un langage abandonné, il avait de bonnes idées pour la manipulation de texte, ces méthodes sont mêmes réimportés dans des langages plus récents :

echo strlen("Bonjour à tous"); // Retourne la longueur de la chaîne de caractères.
echo str_word_count("Bonjour à tous"); // Retourne le nombre de mots dans le string.
echo strrev("Bonjour à tous"); // Inverse les caractères du string.
echo strpos("Bonjour à tous", "à"); // Retourne la position du deuxième string dans le premier.
echo str_replace("monde", "Ricardo", "Salut le monde!");
// Remplace le premier string par le
// deuxième dans le troisième string.

Un truc sympa aussi, puisqu'on travaille avec du web, et que les gens qui utilisent des pages webs sont parfois cons, on a la méthode is_numeric() pour vérifier si une chaîne de caractères peut être représenté comme un nombre, et les engueuler si ce n'est pas le cas. 😉

Value manipulations :
...

Quelques fonctions usuelles pour manipuler des valeurs numériques. Par exemple, on peut récupérer les valeurs maximum en PHP à l'aide de ces nom de variables constantes

Variables constantes
  • PHP_FLOAT_MAX : Le plus grand réel représentable
  • PHP_FLOAT_MIN : Le plus petit réel représentable
  • TO_ASK : Le plus petit nombre réel négatif représentable
  • PHP_FLOAT_DIG : Le nombre maximal de décimaux pouvant être représentés par un float.

Il existe aussi des méthodes utilisables sur des variables, telles que is_float() ou is_double(). On peut aussi tester la validité d'une opération avec la méthode is_nan().

Les méthodes de cast existent aussi, on peut faire un (int) sur un string pour la conversion de cette dernière.

Usage d'un cast

Cast d'un float :

$x = 12.5;
$intX = (int)$x ; // Converti le contenu float de x en entier.
$strY = "15.10" ;
$intY = (int)$strY ; // Converti la chaîne de caractères en entier.
